字符串操作

    <#assign a = "hello"/> <#assign b = "world"/>
  • 连接
    • ${a+b}
  • 截取
    • ${(a+b)?substring(5,9)}
  • 长度
    • ${(a+b)?length}
  • 大写
    • ${(a+b)?upper_case}
  • 小写
    • ${(a+b)?lower_case}
  • index_of
    • ${(a+b)?index_of('l')}
  • last_index_of
    • ${(a+b)?last_index_of('l')}
  • replace
    • ${(a+b)?replace('w','W')}

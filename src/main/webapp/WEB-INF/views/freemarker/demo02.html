逻辑操作

  • 赋值&运算
    • <#assign a=100 /> a= ${a}
    a+100= ${a+100}
  • 对象取值
    • 获取不存在对象的值: ${user2.name!"对象不存在"}
    获取对象存在属性不存在的值: ${user.name2!"值不存在"}
    输出内容:${user.description}
    输出HTML内容:${user.description?html}
  • List循环遍历
    • <#list mylist as item> ${item}
  • Map循环遍历
    • <#list map.keySet() as k> ${k}==${map[k]}
  • 逻辑判断符if
    • <#assign var=100 /> <#if var==99> var 等于99
    <#if var==99> var 等于99 <#else> var 不等于99
    <#if var > 99> var 大于99 <#elseif var==99> var 等于99 <#else> var 小于99
  • 判断List是否为空
    • <#if myList??> myList不为空
    <#list myList as item> ${item} <#else> myList为空
    <#if myList?exists> myList不为空
    <#else> myList为空
  • if多条件||、&&、!
    • <#assign var = "python"> <#if (var=='python' && var?length==6) ||(var=='java')> var is python length is 6 或者 var is java
  • switch case break default
    • <#assign var = 101> <#switch var> <#case 11> <#case 10> var is 11 or 10 <#break> <#case 100> var is 100 <#break> <#default> var is other
    <#assign varStr = "java"> <#switch varStr> <#case "java"> <#case "python"> varStr is java or python <#break> <#case "php"> varStr is php <#break> <#default> varStr is other
